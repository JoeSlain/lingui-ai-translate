# lingui-ai-translate
A node script to translate a lingui file with ai

### Requierments:
A valid OpenAi api key in your env: OPENAI_API_KEY

### Usage
node translator.js <path_to_.po_file> <language>

Example node translator.js locale/de/messages.po de
